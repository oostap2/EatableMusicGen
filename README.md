# EatableMusicGen
Eatable Music Generator is a minimalist procedural audio engine written in Python. It uses algorithmic logic to create an infinite, harmonious stream of music in real-time. By combining a randomized bassline with a calculated melody, it ensures a unique listening experience every time you run it.
# Prerequisites
You will need Python 3 and the `pygame` library installed:
```bash
pip install pygame
```
# Running the Program
Simply execute the script
```bash
python main.py
```
